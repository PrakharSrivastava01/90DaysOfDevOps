End-to-End CI/CD Automation via GitHub CLI
cicd_automation.sh
A single bash script that simulates a complete real-world CI/CD pipeline using the GitHub CLI. It creates a repository, pushes code, opens a pull request, polls for CI check results, and takes action automatically — merging on success or raising an issue on failure. No manual intervention required.

Log Rotation Script
log_rotation.sh
Automatically rotates logs by size or age, archiving old log files to keep directories clean and prevent disk bloat on production systems.

Backup Automation Script
backup.sh
Scheduled backup script that copies critical files and directories to a safe location, compresses them into zip archives for storage efficiency.
🧹
Old Log Cleanup + Zip Archiver
cleanup_logs.sh
Scans for logs older than a defined threshold, cleans them from the active directory, and converts them into compressed zip files for long-term archival.
